核心功能

六大防護能力，一台搞定

Pico-NGFW 100 整合 Lionic DPI 引擎，提供全面的閘道型防護。

防病毒（Anti-Virus）

精準偵測隱藏在不同封包中的病毒，基於 Lionic DPI 深度封包檢測技術，即時攔截各類型惡意軟體。

防駭客入侵（Anti-Intrusion IPS）

保護終端設備及 IoT 設備，即時偵測並阻擋針對網路的駭客入侵行為，減少 CVE 漏洞被利用的風險。

防惡意網站（Anti-WebThreat）

自動阻擋釣魚網站、勒索網站及已知 C&C 命令控制伺服器，防止用戶誤入惡意頁面造成資料外洩。

地理封鎖（Geo-IP Block）

封鎖指定國家/地區來源的流量，根據業務需求自訂封鎖規則，大幅降低來自高風險地區的攻擊。

OS 漏洞防護

無需更新系統即可防護老舊 Windows / Linux 系統漏洞，守護仍在使用過期 OS 的設備，減少資安風險。

CMS 中央管理

遠端集中管理多台設備，設備異常即時 LINE 威脅通知，支援週報表、威脅百科，讓管理輕鬆有效率。

部署場景

適用於各類中小型機構

真實客戶案例

Pico-NGFW 100 已在全台數千家連鎖商店、餐廳與公家機關中穩定運行，守護各類環境的網路資安。

  • 連鎖餐飲 — 全台多分點統一部署，CMS 集中管理，IT 人員大幅減少維護時間
  • 大眾運輸 — 邊緣部署方案，防護公共 Wi-Fi 環境，阻擋大量惡意掃描
  • 連鎖藥妝 — Gateway 方案，防護 POS 系統與 IoT 設備，符合金融業資安規範
