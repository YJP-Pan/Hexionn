基於 Lionic DPI 核心技術，Lionfilter 200 提供企業與電信業者最精準的 URL 過濾、應用程式識別與 QoS 頻寬管理，已授權全球多家知名 ISP 業者採用。

核心功能

精準過濾，全球信賴

Lionfilter 200 以 Lionic DPI 核心技術驅動，提供全球最精準的網路內容管控能力。

URL 過濾

每日自動更新逾 5 億條 URL 黑名單資料，分類精準涵蓋成人、賭博、惡意軟體、釣魚等 80+ 類別，是業界最大規模 URL 資料庫之一。

應用程式識別管控

精準識別 1000+ 應用程式，包含 Netflix、YouTube、P2P 下載、社群媒體、線上遊戲等，可按應用程式類型設定允許、封鎖或限速策略。

QoS 頻寬管理

依應用程式類型、用戶群組、時段靈活設定頻寬優先順序，確保關鍵業務應用獲得充足頻寬，同時限制非業務流量佔用。

家長管制（Parental Control）

深度分類過濾功能，適合校園、家庭及企業部署，有效阻擋不適合年齡層的內容，支援時段管制與分組政策。

Web Guard 惡意網站防護

即時封鎖釣魚網站、勒索軟體下載頁面、C&C 命令控制伺服器及已知惡意網址，每日更新威脅資料庫，保護用戶免受 Web 層攻擊。

自訂政策引擎

彈性設定分類規則與白名單／黑名單，支援 API 整合與 SDK 授權部署，符合各國法規（CIPA、GDPR、電信業者自主管理法規）需求。

ISP、企業、校園全覆蓋

適用場景

  • ISP / 電信業者 — 在 ISP 出口閘道部署，提供用戶家長管制服務，符合在地法規要求，增加 ISP 加值服務收入
  • 企業網路管理 — 精細管控員工上網行為，按部門設定差異化政策，防止資料外洩與生產力浪費
  • 校園網路安全 — 依年齡分組管控，符合 CIPA 等校園網路安全法規，保護學生免受有害內容影響
  • SDK / API 整合 — 授權模式提供 SDK 與 API，可整合至現有網路設備或雲端平台，快速上市
